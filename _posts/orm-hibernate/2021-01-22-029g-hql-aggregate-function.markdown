Hai, pada kesempatan kali ini saya mau membahas tentang Aggregation Function dengan menggunakan HQL. Aggregation pada dasarnya ada beberapa function yang kita bisa gunakan yaitu 

1. `COUNT`
2. `AVG`
3. `MIN`
4. `MAX`
5. `SUM`
