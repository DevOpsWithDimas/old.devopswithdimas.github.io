Data Manipulation atau singkatan dari DML yaitu sebuah statement yang digunakan untuk memanipulasi data dalam sebuah tabel di Database. Perintah DML terdiri dari 3 bagian yaitu

1. Insert Data
2. Update Data
3. Delete Data
